🎓 CampusLive – Smart Campus Dashboard

CampusLive is a real-time campus monitoring dashboard that helps students track availability, crowd levels, and live updates across campus facilities.

🚀 Features
🧪 Live status of labs, printers, library seats
🍽️ Canteen crowd insights & wait time
👨‍🏫 Faculty availability tracking
📡 Real-time activity feed (auto-updating)
🏆 Top contributors leaderboard
🔔 Alerts & reporting system
🗺️ Campus navigation & analytics
